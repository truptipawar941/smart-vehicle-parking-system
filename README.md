# Smart Parking System

## Overview
This project is an IoT-based Smart Parking System developed using ESP8266 and Ultrasonic Sensors.

## Features
- Detects vehicle presence in parking slots
- Displays slot status (Empty/Full)
- Shows welcome and parking messages
- Helps manage parking efficiently

## Components Used
- ESP8266
- Ultrasonic Sensors
- LCD Display
- Servo Motor (if used)
- LEDs

## Software Used
- Arduino IDE
- Wokwi
